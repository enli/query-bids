In Developer console,

var bids = new BidsGenerator().generate(); // generate() also accepts a number

var q = new BidsQuery(bids); // instantiate BidsQuery

q.getBidsByLastUpdated(5); //get latest 5 bids by updated date

q.getBidsByLastExecuted(5); //get latest 5 bids by last executed date

Bonus:

getBidsByLastUpdated and getBidsByLastExecuted methods accept filter name as second parameter.

q.getBidsByLastUpdated(5, 'foo'); // filter foo tag
